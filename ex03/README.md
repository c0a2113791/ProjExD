### memo
- ブランチ関連のコマンド
- [ ] ブランチ作成するコマンド
git branch ブランチ名
- [ ] ブランチを切り替えるコマンド
git swith ブランチ名
- [ ] ブランチをマージするコマンド
git merge ブランチ名
- [ ] マージ完了後、別のブランチを削除するコマンド
git branch -d ブランチ名

ブランチを作成し作業を行うのはメインで行っているプログラムに影響を与えず、目的別で同時並行で作業が行えるためである。
また、メインとは別のブランチで作業が完了した際はメインに戻り変更履歴を取り込む（マージ）
